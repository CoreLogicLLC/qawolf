<Syntax code="register(context)" />

Register a [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance with QA Wolf, which:

- Registers the [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) so [`waitForPage`](wait_for_page) works
- Creates [debug artifacts](../../run_tests_in_ci#debug) when [QAW_ARTIFACT_PATH](../environment_variables#qaw_artifact_path) is specified

<ArgumentRegister />

### Returns

<ReturnVoid />

### Examples

Give QA Wolf access to the [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance that your test is running against.

```js
const qawolf = require('qawolf');
// ...
qawolf.register(context);
```
