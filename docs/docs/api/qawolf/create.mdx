<Syntax code="create()" />

Convert your browser actions to code. Call `create` in a test file to insert new code where it was called.

See the [guide on editing tests](../../edit_a_test) to learn more about using `create`.

### Arguments

No arguments.

### Returns

<ReturnVoid />

### Examples

Create new code in an existing file. You must call [`qawolf.register(context)`](register) first so that QA Wolf has access to the [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance.

```js
const qawolf = require('qawolf');
// ...
// somewhere earlier in file
qawolf.register(context);

await qawolf.create();
```
